Vision Core

School exhibition project by Keivan Asadi & Niranjan M

A Python program capable of tracking the user's face, hands and blinks to play and interact with custom tools; created via MediaPipe.
Face Tracking

Face Mesh

While this wasn't necessarily needed, I still thought it was really cool being able to turn your face into a digital mesh.

Like a lot of the apps below, this was made possible with MediaPipe.

Blink Detection

Using MediaPipe face tracking, we can detect eye landmarks each frame and detect blinks.

Flappy Bird

Using blink detection, we can now play Flappy Bird just by blinking.

Hand Tracking

Hand Mesh

MediaPipe allows full hand tracking. It can even track full body pose systems, but this project focuses on hands and face.

Painting

Using hand gestures, we can draw in real time.
Modes: Paint ☝️, Hover ✌️, Erase ✋

But art was never my talent 😶‍🌫️
Face Drawing

A drawing canvas projected onto the user's face using face tracking.

This feature was rushed, hence the imperfections.

Setup

To use this program yourself, you first need to install MediaPipe and OpenCV. 


pip install mediapipe opencv-python pygame
Make sure you use Python 3.10 to get this working.
