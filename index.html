Vision Core

School exhibition project by Keivan Asadi & Niranjan M

A Python program capable of tracking the user's face, hands and blinks to play and interact with custom tools; created via MediaPipe.
Face Tracking

Face Mesh

While this wasn't necessarily needed, i still thought it was really cool being able to turn your face into a digital mesh.

Like a lot of the apps you'll see below, this was made possible with MediaPipe.

Blink Detection

Now's where the fun begins; using MediaPipe's face tracking, we can track the individual points around the user's eye each frame to know when they blink.

Flappy Bird

Using blink detection, we can now play flappy bird by just blinking!

Hand Tracking

Hand Mesh

We can also track the user's hand using MediaPipe.

In fact, it's possible to track an entire body using MediaPipe, but you wont be seeing that here.

Painting

Now, using hand tracking, we can paint on the screen!

There are three modes; Paint (☝️), Hover (✌️) and Erase (✋).

Using these modes, we can create more complicated creations as seen below.

But art was never my talent 😶‍🌫️
Face Drawing

To put face tracking to use, we can have a canvas for the user to draw on, to which it will be projected on the user's face.

This feature was a little rushed, hence why it doesnt look that great.
